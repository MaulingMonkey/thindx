[docs.microsoft.com] IsWindow

Determines whether the specified window handle identifies an existing window.

Valid uses of this function are few and far between - windows belonging to another thread or process could be destroyed immediately after this returns true, and invalidated handles might suddenly spring back to life as the handle value is reused.

Example

assert!( win32::is_window(  valid_hwnd));
assert!(!win32::is_window(invalid_hwnd));
assert!(!win32::is_window(std::ptr::null_mut()));
